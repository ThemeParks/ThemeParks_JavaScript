# themeparks

The JavaScript/TypeScript SDK for the ThemeParks.wiki API.

**v7 is under active development.** A full README will land with the v7.0.0 release.
For the current stable release (v6), install `themeparks` pinned to `<7` — note that v6 is raw OpenAPI Generator output and v7 is a hand-crafted TypeScript SDK with proper types and ergonomic helpers.

- API documentation: https://api.themeparks.wiki
- Repository: https://github.com/ThemeParks/ThemeParks_JavaScript
